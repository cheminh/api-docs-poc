# TripX Distribution API Documentation

Welcome to the TripX Distribution API documentation. This site provides comprehensive API reference and usage guides.

## Overview

The TripX Distribution API allows you to integrate travel booking functionality into your applications. Access flight search, booking management, and more through our REST API.

## API Reference

View the complete [API Reference](/docs/api) with interactive examples and detailed endpoint documentation.

## Getting Started

1. **Get an API Key** - Sign up at [dashboard.tripx.com](https://dashboard.tripx.com)
2. **Read the Guide** - Check out [Getting Started](./getting-started)
3. **Explore the API** - Try the [API Reference](/docs/api)
